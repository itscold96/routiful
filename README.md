## 프로젝트 개요
### 세트 수 늘렸더니, 몇 세트까지 했는지 기억이 안나서 만든 개인 프로젝트
- **Routiful: 고작 6세트했다, 아직 4세트 남았다.**
- [서비스 바로가기](https://routiful.vercel.app/)

### 세트 수 늘렸더니, 몇 세트까지 했는지 기억이 안나서 만든 개인 프로젝트

- 📓 **소켓 통신**을 이용한 친구와의 **공유 그림일기**
- 🙋 **사용자 관점**의 편리함을 고려한 **직관적**인 인터페이스와 애니메이션
